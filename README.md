## Install with Npm or Yarn

```bash
npm install multichain-utility
```

```bash
yarn add multichain-utility
```

## Install multichain-utility with npm link

```
git clone <this repo>
cd hardhat-test-utility
npm install
npm link

cd <your project>
npm link hardhat-test-utility
```
